# DevsWorld

A social media clone full stack MERN web app to connect with like minded developers.

## Features

- Signup/Login
- Create/Delete a Post
- Upload Images
- Like/Unlike a Post
- Add or delete comments
- Follow/Unfollow users
- General Feed and Your Feed
- Search Users
- Edit user profile

## Tech Stack

- React
- React-Router-Dom
- Reduxjs-Toolkit
- Styled Components
- Nodejs
- Express
- MongoDB
